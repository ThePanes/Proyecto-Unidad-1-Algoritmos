Instrucciones para ejecutar y/o compilar el programa:

1. Debe contar con el programa DEV C++ 4.9.9.2 o similar, o Codeblocks en su ultima version de ser posible.
2. Clonar o descargar el repositorio de github en su ordenador y abrir la carpeta contenedora de los archivos.
3. Abrir el archivo proyectoUnidad1.c
4. Para poder compilar el archivo, primero debemos asegurar de que en la misma carpeta de este proyecto este el archivo "guardians.txt" que debe contener tipos de dato de este tipo: Gandalf, Mage,40,20,10 para poder realizar la correcta lectura, tenga en cuenta que dependiendo del orden influye la vida ataque y defensa de la carta, por lo que se debe tener precaución para la lectura de datos.
5. Abrir el archivo proyectoUnidad1.c con su interfaz de trabajo de preferencia, en el caso personal, utilice CodeBlocks.
6. Sí se aseguró de que el archivo "guardians.txt" está dentro de la misma carpeta que el archivo proyectoUnidad1.c, entonces puede compilar con precaución.
7. Sí desea introducir un archivo personalizado en el formato .txt que funcione dentro del código, tendrá que dirigirse a la función "void initFromText" que abre el archivo, y pasarle el nuevo nombre del documento que desea que el programa abra para su lectura de datos en la variable char *archivito = "guardians.txt"; en donde "guardians.txt" deberá ser reemplazado por el nombre del archivo del cual desea la lectura, recordar tener en cuenta que tiene que ser en formato .txt para su correcto funcionamiento y que este deberá estar exactamente en la misma carpeta que proyectoUnidad1.c
8. Compilar y ejecutar el programa con su compilador de preferencia, no necesita pasarle parámetros extra para ejecutar, solo ejecutar asegurándose de que todos los archivos estén en la misma carpeta.
